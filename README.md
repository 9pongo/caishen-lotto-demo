# 財神樂透 — 試玩版（純前端 DEMO）

手機直式試玩：**https://9pongo.github.io/caishen-lotto-demo/**

- 純前端展示版：瀏覽器內建假伺服器（固定腳本），**首輪依序展示全部 24 格結果**
  （第一抽即特等賞演出），之後循環有代表性的組合。
- **非正式機率、不代表 RTP** — 正式版一切抽獎結果由伺服器（CSPRNG）決定，前端只是播放器。
- 操作：許願 1 次 / 許願 5 次；×3 = 三倍轉數；「跳過」開啟時許願後直接顯示結算卡（預設 OFF，會完整播跑燈演出）。
  右上選單＝遊戲說明（含獎池條款）與測試加值。
- 版型：SKIN_SPEC v1.1（720×1280 直式）。

> 開發倉庫（數學引擎 / Orleans 伺服器 / 後台）為私有；本倉庫僅供試玩溝通。

## 美術資源（skin/）

`skin/*.png` 一律走「檔名 = 元件 id 把 `.` 換成 `-`」的規則，開機時 HEAD 探測，
**有檔就貼圖、沒檔就退回佔位繪圖**，換圖不用改程式。座標一律吃 `skin/layout.json`。

已切好（取自美術稿 1438×2976）：

| 檔案 | 用途 | layout 元件 |
|---|---|---|
| `bg.png` | 背景全圖：頂端標題美術字、金幣裝飾、底部金幣堆與狀態列 | — |
| `board-frame.png` | 盤面外框 + 中央紅卡（內容淨空，格子與文字由程式畫） | `board.frame` |
| `deco-title.png` | 中央「財神樂透」緞帶 | `deco.title` |
| `deco-mascot.png` | 財神 | `deco.mascot` |
| `deco-ingots.png` | 底部金幣堆 | `deco.ingots` |

尚未替換（仍是程式繪圖）：24 格格子、許願／×3／跳過按鈕、右上選單、結算卡。
格子與按鈕要換圖得先有**不含文字的底圖**（文字由程式即時畫，會疊字）；
`cell.value` 目前只吃單一張圖，三色循環要換圖的話得先擴成 `cell.value.0/1/2`。

底部狀態列說明：

- 左下角牌子＝帳號 ID（上）＋ 餘額（下），由 `hud.player` / `hud.balance` 定位，
  數字是程式即時畫的（`bg.png` 裡那格已洗掉印死的數字）。
- 右側四顆功能鍵（音效／說明／紀錄／大廳）**目前只是 `bg.png` 的底圖，尚未接功能**。
- 加值與遊戲說明仍在右上角的選單鍵。
